ToLoG (C++)

Tobis Library of Geometry is an ongoing project where I put some general purpose datastructures and algorithms that I use in other projects.

Wordhunter (Godot)

A simple game made in Godot where the goal is to find words in a hexagonal grid of letters. I have not implemented any losing condition ;)

Happy Little Trains (Unity)

A team project for the course "Design and Graphics Programming for Game Development" at the university of Fribourg. For a semester, two friends and I developed this little puzzle game where you need to guide colored trains to their matching stations.

Cutting Planes 2D Visualizer (C++, Python)

Cutting Planes is an algorithm to solve mixed integer linear problems. This project contains a c++ solver (for arbitrary dimensions), and a visualization tool made in python (for 2d problems). It was created for the Applied Optimization Seminar at the Unversity of Bern in the spring semester 2024.

Interpreter (Java)

From text to tokens to a tree, here I implemented an interpreter for a custom programming language. This includes different variable types, operations, loops, scopes, and functions.

Logic Lounge (Java)

The project includes some simple algorithms related to boolean formulas like creating a Karnaugh-Veitch diagram, showing the steps of the Quine-McCluskey algorithm to minimize boolean functions, or an algorithm for computational tree logic. All of this is provided with a nice interface where you can type your formulas. The output can be either stored as an image, or as latex source code.

Raytracer (Java)

An editor to raytrace your own little scenes and produce (hopefully beautiful) images. All the logic is written in good ol' java. You can define primitive shapes, load meshes, texture- and normal-maps, define certain material properties of your shapes, and more....

Sokoban (Java)

A classic where you push boxes (or flies?) onto buttons (or into spider webs?). The game automatically reads levels from the corresponding folder, so take a look at some level files, and create your own if you like :)

Santa Shooter (JavaScript)

Once upon a December, there were no cookies for me. To cope with this, I developed this shoot 'em up game using GameMaker Studio in which Santa has to defend himself against the approaching cookies. Recently I had the desire to reimplement the game with its original assets using JavaScript.
